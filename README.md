# 🌍 TravelNest

A premium, responsive travel booking platform built with **React 19** and **Tailwind CSS**.

---

## ✨ Key Features

- **Real-time Search:** Debounced global search across destinations and tours.  
- **Smooth UX:** Custom animations, sticky navigation, and a multi-step booking modal.  
- **Dynamic Routing:** Comprehensive detail pages for destinations and experiences.  
- **Responsive:** Mobile-first design using Tailwind utility classes.  

---

## 🛠️ Tech Stack

- **Frontend:** React 19, React Router 7  
- **Styling:** Tailwind CSS 4, FontAwesome 6  
- **State Management:** React Hooks (`useState`, `useEffect`, `useRef`)  

---

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm start

# Build for production
npm run build
